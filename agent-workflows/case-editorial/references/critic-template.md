# Critic Review Template

Use this as the critic agent's checklist:

- Does every factual claim in the draft map to a frozen quote?
- Are there any inferred deaths, injuries, or accountability claims?
- Are any source URLs too generic or homepage-level?
- Did the draft preserve nulls for unresolved conflicts?
- Did the rewrite introduce new facts or new framing?
- Is the case identity still the same incident after the rewrite?
- Does each populated field have a quote-backed evidence entry?

Return only:
- verdict
- rejected_claims
- required_fixes
- notes
